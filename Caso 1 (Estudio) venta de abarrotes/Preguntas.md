# Caso de Estudio 1: Control de Ventas por Categoría en una Tienda de Abarrotes

## Preguntas de reflexión

### 1. ¿Por qué es útil analizar las ventas por categoría y no solo las ventas totales?

Analizar las ventas por categoría permite identificar el desempeño individual de cada grupo de productos. Esto facilita conocer cuáles categorías generan mayores ingresos y cuáles requieren estrategias de mejora, información que no se puede obtener observando únicamente el total general de ventas.

### 2. ¿Qué implica que una categoría tenga ventas críticas?

Que una categoría presente ventas críticas significa que su nivel de ventas es considerablemente menor en comparación con las demás categorías. Esta situación puede indicar una baja demanda de los productos, problemas de promoción o la necesidad de implementar acciones que contribuyan a incrementar sus ventas.

### 3. ¿Cómo cambiaría la lógica si el vector representara tiendas en vez de categorías?

La lógica del programa se mantendría prácticamente igual, ya que seguiría utilizándose un vector para almacenar nombres y otro para almacenar valores. Sin embargo, el análisis se enfocaría en comparar el rendimiento de cada tienda en lugar de las categorías de productos, permitiendo evaluar cuáles sucursales presentan mayores o menores niveles de ventas.
